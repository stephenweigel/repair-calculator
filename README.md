# Repair Calculator Application
This is an application for helping quote customers at a computer repair shop.

## Instructions
1. Make sure you have [node.js](http://nodejs.org/) and [git](http://git-scm.com/) installed on your machine.
2. Clone this into your local machine using the terminal (mac) or Gitbash (PC) `> git clone https://github.com/stephenweigel/repair-calculator.git`
3. CD to the folder `cd repair-calculator`
4. Run `> npm-install` to install the project dependencies
5. Install gulp.js via the Mac terminal or Gitbash on a PC `> npm install -g gulp`
5. Run the Gulp command `> gulp`
6. Fire up your browser at `http://localhost:8080`
